LibGDX

LibGDX is an open source, cross-platform, and most importantly free application programming interface that allows you to easily get started on creating your own video game.

Tutorials on LibGDX

These are very helpful tutorials on how to get started using LibGDX as well as the many features that it provides.

Eclipse

Eclipse is an integrated development environment that will allow you to code in Java and a variety of other languages. LibGDX uses Java to run its programs. Since Eclipse is an IDE, implementation of code is a lot easier than using a general text editor. Eclipse also allows you to view errors at compile time so that you know where you made mistakes.

GIMP | Tutorials

If you're not interested in hiring an artist for your video game and plan to draw everything yourself, GIMP is the way to go! GIMP is a free image manipulation program similar to Photoshop. It allows you to create images and alter them to the way you see fit. GIMP's interface is easy to use and allows beginners to pick up on its features in minimal time.
